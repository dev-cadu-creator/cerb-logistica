# 📦 App CERB - Logística e Gestão de Entregas

Este projeto nasceu da necessidade real de organizar a minha rotina como motorista e estafeta em Uberlândia-MG. O **App CERB** ajuda a gerir rotas, ganhos e despesas de manutenção, transformando desafios do dia a dia em soluções tecnológicas.

## 🚀 Sobre o Projeto

O aplicativo foi idealizado enquanto eu estudava a lógica de programação em **Java**. Percebi que poderia aplicar o que estava a aprender na faculdade de **Ciência da Computação** para otimizar o meu trabalho com entregas (Mercado Livre/Shopee) e transporte de passageiros.

### Principais Funcionalidades:
- **Controlo de Ganhos:** Registo rápido de valores recebidos por plataforma.
- **Gestão de Despesas:** Monitorização de gastos com combustível e manutenção do meu Voyage G8.
- **Logística de Rotas:** Organização das paragens para entregas de forma mais eficiente.
- **Interface Intuitiva:** Desenvolvida para ser usada de forma prática entre uma corrida e outra.

## 🛠️ Tecnologias Utilizadas

Para este projeto, utilizei uma combinação de tecnologias para garantir uma base sólida e uma interface amigável:

* **Lógica e Estrutura:** Java
* **Interface (Front-end):** HTML5, CSS3 e JavaScript
* **Estilo:** Design focado na usabilidade "on-the-road".
